# AIALL 使用指南

## 💬 如何提问

1. 打开 App，在底部输入框输入你的问题
2. 点击发送按钮或按回车键
3. 多个 AI 模型同时回答，横向滑动查看不同模型的答案

## 🔄 浏览答案

- **滑动切换**: 在模型答案区域左右滑动
- **点击标签**: 顶部模型标签栏点击可快速跳转
- **复制答案**: 每个答案卡片右上角有复制按钮

## 🔥 热点提问

- 首页展示今日热点（百度/微博/知乎来源）
- **点击热点标题** 直接发送该问题
- **下拉页面** 刷新获取最新热点

## ⚙️ 模型管理

1. 点击底部 Tab 进入「模型管理」
2. 点击预设模型的「配置」按钮，填入 API Key
3. **下拉页面** 测试所有模型连接状态
4. 彩色 = 正常，灰色 = 异常

## 🌙 夜间模式

- 在模型管理页右上角切换：☀ 浅色 / 🌙 深色 / A 跟随系统
- 选择自动跟随系统时，App 会根据手机系统设置自动切换
- 主题选择会自动保存，下次打开保持上次设置

## 📋 历史记录

- 所有对话自动保存在「历史记录」Tab
- 底部显示模型数量和对话轮数统计
- 长按或点击删除图标可删除记录

## 📖 查看帮助

- **首次安装**: 自动弹出使用指南
- **随时查看**: 双击首页"AIALL"文字可再次打开

## 💡 小技巧

- 横向滑动答案卡片时，顶部标签栏会自动高亮当前模型
- 热点标题会实时更新，下拉可获取最新内容
- 模型管理页支持拖拽排序，长按可删除
- 启动时会自动检测所有已配置模型的连接状态
